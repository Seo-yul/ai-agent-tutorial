# AI Agent 개발 튜토리얼

MCP와 LangGraph를 Python으로 배우는 실전 가이드.

## 실행

```bash
cd /Users/a10619/tmp/ai-agent-tutorial
python3 -m http.server 8899
```

브라우저에서 http://localhost:8899 접속.

## 구성

| 코스 | 챕터 | 내용 |
|------|------|------|
| MCP | 13개 (기본 8 + 심화 5) | Model Context Protocol — 도구 서버 구축 |
| LangGraph | 12개 (기본 7 + 심화 5) | 그래프 기반 AI 에이전트 구축 |
