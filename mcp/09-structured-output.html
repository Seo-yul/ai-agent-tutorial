Structured Output

Pydantic лӘЁлҚёлЎң нғҖмһ… м•Ҳм „н•ң лҸ„кө¬ м¶ңл Ҙмқ„ м •мқҳн•ҳкі , н”„лЎңнҶ мҪң мҲҳмӨҖм—җм„ң мҠӨнӮӨл§Ҳк°Җ м–ҙл–»кІҢ лҸҷмһ‘н•ҳлҠ”м§Җ мқҙн•ҙн•ңлӢӨ

Structured OutputмқҙлһҖ?

н•өмӢ¬ к°ңл…җ: кө¬мЎ°нҷ”лҗң м¶ңл Ҙ

MCP лҸ„кө¬лҠ” кё°ліём ҒмңјлЎң TextContent(л¬ёмһҗм—ҙ)лҘј л°ҳнҷҳн•ңлӢӨ. Structured OutputмқҖ лҸ„кө¬к°Җ мӮ¬м „м—җ м •мқҳлҗң JSON Schemaм—җ л§һлҠ” кө¬мЎ°нҷ”лҗң лҚ°мқҙн„°лҘј л°ҳнҷҳн•ҳлҠ” нҢЁн„ҙмқҙлӢӨ. нҒҙлқјмқҙм–ёнҠё(LLM)лҠ” лҸ„кө¬лҘј нҳём¶ңн•ҳкё° м „м—җ л°ҳнҷҳк°’мқҳ кө¬мЎ°лҘј м•Ң мҲҳ мһҲмңјлҜҖлЎң, кІ°кіјлҘј нҢҢмӢұн•ҳм—¬ нӣ„мҶҚ мһ‘м—…м—җ мҰүмӢң нҷңмҡ©н•  мҲҳ мһҲлӢӨ.

мҷң кө¬мЎ°нҷ”лҗң м¶ңл Ҙмқҙ н•„мҡ”н•ңк°Җ? м„ё к°Җм§Җ н•өмӢ¬ мқҙмң к°Җ мһҲлӢӨ:

  • нҢҢмӢұ м•Ҳм •м„ұ — мһҗмң  нҳ•мӢқ н…ҚмҠӨнҠёлҘј м •к·ңмӢқмңјлЎң нҢҢмӢұн•ҳлҠ” кІғмқҖ м·Ём•Ҫн•ҳлӢӨ. JSON SchemaлЎң кө¬мЎ°к°Җ ліҙмһҘлҗҳл©ҙ нҢҢмӢұ мӢӨнҢЁк°Җ мӣҗмІңм ҒмңјлЎң м°ЁлӢЁлҗңлӢӨ.
  • нғҖмһ… м•Ҳм „м„ұ — temperatureк°Җ floatмқҙлқјлҠ” кІғмқҙ мҠӨнӮӨл§Ҳ мҲҳмӨҖм—җм„ң ліҙмһҘлҗҳлҜҖлЎң, нҒҙлқјмқҙм–ёнҠё мҪ”л“ңм—җм„ң нҳ• ліҖнҷҳ мҳӨлҘҳк°Җ л°ңмғқн•ҳм§Җ м•ҠлҠ”лӢӨ.
  • мһҗлҸҷ л¬ём„ңнҷ” — Field(description=...)мңјлЎң мһ‘м„ұн•ң м„ӨлӘ…мқҙ JSON Schemaм—җ нҸ¬н•Ёлҗҳм–ҙ, лі„лҸ„ л¬ём„ң м—ҶмқҙлҸ„ LLMмқҙ к°Ғ н•„л“ңмқҳ мқҳлҜёлҘј м •нҷ•нһҲ нҢҢм•…н•ңлӢӨ.
INFO

Structured OutputмқҖ MCP 2025-03-26 мҠӨнҺҷм—җм„ң лҸ„мһ…лҗҳм—ҲлӢӨ. FastMCPлҠ” лҸ„кө¬ н•ЁмҲҳмқҳ л°ҳнҷҳ нғҖмһ… нһҢнҠёлҘј л¶„м„қн•ҳм—¬ мһҗлҸҷмңјлЎң outputSchemaлҘј мғқм„ұн•ҳкі , л°ҳнҷҳк°’мқ„ structuredContentлЎң м§Ғл ¬нҷ”н•ңлӢӨ.

н”„лЎңнҶ мҪң мҲҳмӨҖмқҳ лҸҷмһ‘

Structured Outputмқҙ MCP н”„лЎңнҶ мҪңм—җм„ң м–ҙл–»кІҢ лҸҷмһ‘н•ҳлҠ”м§Җ лӢЁкі„лі„лЎң мӮҙнҺҙліҙмһҗ. н•өмӢ¬мқҖ outputSchemaк°Җ лҸ„кө¬ л©”нғҖлҚ°мқҙн„°м—җ нҸ¬н•Ёлҗҳкі , нҳём¶ң кІ°кіјк°Җ structuredContentлЎң л°ҳнҷҳлҗңлӢӨлҠ” кІғмқҙлӢӨ.

1лӢЁкі„: tools/listм—җм„ң outputSchema л…ём¶ң

нҒҙлқјмқҙм–ёнҠёк°Җ tools/listлҘј мҡ”мІӯн•ҳл©ҙ, м„ңлІ„лҠ” к°Ғ лҸ„кө¬мқҳ outputSchemaлҘј мқ‘лӢөм—җ нҸ¬н•Ён•ңлӢӨ:

→ REQUEST — tools/list
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "tools/list"
}
← RESPONSE
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": {
    "tools": [
      {
        "name": "get_weather",
        "description": "лҸ„мӢңмқҳ лӮ м”ЁлҘј мЎ°нҡҢн•ңлӢӨ",
        "inputSchema": {
          "type": "object",
          "properties": {
            "city": { "type": "string" }
          },
          "required": ["city"]
        },
        "outputSchema": {
          "type": "object",
          "title": "WeatherData",
          "description": "лӮ м”Ё м •ліҙ",
          "properties": {
            "temperature": { "type": "number", "description": "м„ӯм”Ё мҳЁлҸ„" },
            "humidity": { "type": "number", "description": "мҠөлҸ„ (%)" },
            "condition": { "type": "string", "description": "лӮ м”Ё мғҒнғң" },
            "wind_speed": { "type": "number", "description": "н’ҚмҶҚ (m/s)" }
          },
          "required": ["temperature", "humidity", "condition", "wind_speed"]
        }
      }
    ]
  }
}

2лӢЁкі„: tools/callм—җм„ң structuredContent л°ҳнҷҳ

лҸ„кө¬лҘј нҳём¶ңн•ҳл©ҙ, м„ңлІ„лҠ” content(TextContent)мҷҖ structuredContentлҘј лҸҷмӢңм—җ л°ҳнҷҳн•ңлӢӨ:

→ REQUEST — tools/call
{
  "jsonrpc": "2.0",
  "id": 2,
  "method": "tools/call",
  "params": {
    "name": "get_weather",
    "arguments": { "city": "м„ңмҡё" }
  }
}
← RESPONSE
{
  "jsonrpc": "2.0",
  "id": 2,
  "result": {
    "content": [
      {
        "type": "text",
        "text": "{\"temperature\": 22.5, \"humidity\": 45.0, \"condition\": \"л§‘мқҢ\", \"wind_speed\": 5.2}"
      }
    ],
    "structuredContent": {
      "temperature": 22.5,
      "humidity": 45.0,
      "condition": "л§‘мқҢ",
      "wind_speed": 5.2
    }
  }
}
Deep Dive: contentмҷҖ structuredContentмқҳ лі‘н–ү л°ҳнҷҳ

structuredContentк°Җ мһҲлҚ”лқјлҸ„ content(TextContent)лҠ” н•ӯмғҒ н•Ёк»ҳ л°ҳнҷҳлҗңлӢӨ. мқҙлҠ” н•ҳмң„ нҳёнҷҳм„ұмқ„ мң„н•ң м„Өкі„мқҙлӢӨ. outputSchemaлҘј мқҙн•ҙн•ҳм§Җ лӘ»н•ҳлҠ” кө¬нҳ• нҒҙлқјмқҙм–ёнҠёлҸ„ contentмқҳ н…ҚмҠӨнҠёлҘј мқҪмқ„ мҲҳ мһҲлӢӨ. л°ҳл©ҙ, outputSchemaлҘј м§Җмӣҗн•ҳлҠ” нҒҙлқјмқҙм–ёнҠёлҠ” structuredContentлҘј нҢҢмӢұн•ҳм—¬ нғҖмһ… м•Ҳм „н•ҳкІҢ кІ°кіјлҘј мІҳлҰ¬н•  мҲҳ мһҲлӢӨ.

м „мІҙ нқҗлҰ„ лӢӨмқҙм–ҙк·ёлһЁ

sequenceDiagram
    participant C as Client (LLM)
    participant S as MCP Server

    C->>S: tools/list
    S-->>C: outputSchema нҸ¬н•Ёлҗң лҸ„кө¬ лӘ©лЎқ
    Note over C: outputSchemaлЎң л°ҳнҷҳк°’ кө¬мЎ° нҢҢм•…

    C->>S: tools/call (get_weather, city="м„ңмҡё")
    Note over S: Pydantic лӘЁлҚё мғқм„ұ + кІҖмҰқ
    S-->>C: content (TextContent) + structuredContent (JSON)
    Note over C: structuredContentлҘј нҢҢмӢұн•ҳм—¬ нҷңмҡ©

Pydantic лӘЁлҚё мӮ¬мҡ©

к°ҖмһҘ к¶ҢмһҘлҗҳлҠ” л°©лІ•мқҖ Pydantic BaseModelмқ„ мӮ¬мҡ©н•ҳлҠ” кІғмқҙлӢӨ. Field(description=...)мңјлЎң к°Ғ н•„л“ңм—җ м„ӨлӘ…мқ„ л¶ҷмқҙл©ҙ JSON Schemaм—җ нҸ¬н•Ёлҗҳм–ҙ LLMмқҙ кІ°кіјлҘј м •нҷ•нһҲ мқҙн•ҙн•ңлӢӨ.

кё°ліё лӘЁлҚё м •мқҳ

from pydantic import BaseModel, Field
from mcp.server.fastmcp import FastMCP

mcp = FastMCP("Structured Output Example")


class WeatherData(BaseModel):
    """лӮ м”Ё м •ліҙ"""
    temperature: float = Field(description="м„ӯм”Ё мҳЁлҸ„")
    humidity: float = Field(description="мҠөлҸ„ (%)")
    condition: str = Field(description="лӮ м”Ё мғҒнғң (л§‘мқҢ, нқҗлҰј, л№„ л“ұ)")
    wind_speed: float = Field(description="н’ҚмҶҚ (m/s)")


@mcp.tool()
def get_weather(city: str) -> WeatherData:
    """лҸ„мӢңмқҳ лӮ м”ЁлҘј мЎ°нҡҢн•ңлӢӨ"""
    return WeatherData(
        temperature=22.5,
        humidity=45.0,
        condition="л§‘мқҢ",
        wind_speed=5.2,
    )

мӨ‘мІ© лӘЁлҚё

Pydantic лӘЁлҚёмқ„ лӢӨлҘё лӘЁлҚёмқҳ н•„л“ңлЎң мӮ¬мҡ©н•ҳл©ҙ ліөмһЎн•ң кө¬мЎ°лҸ„ мһҗм—°мҠӨлҹҪкІҢ н‘ңнҳ„лҗңлӢӨ. JSON Schemaм—җ мӨ‘мІ© кө¬мЎ°к°Җ мһҗлҸҷмңјлЎң л°ҳмҳҒлҗңлӢӨ.

class Coordinates(BaseModel):
    """мўҢн‘ң м •ліҙ"""
    latitude: float = Field(description="мң„лҸ„", ge=-90, le=90)
    longitude: float = Field(description="кІҪлҸ„", ge=-180, le=180)


class ForecastItem(BaseModel):
    """мӢңк°„лі„ мҳҲліҙ"""
    time: str = Field(description="мҳҲліҙ мӢңк°Ғ (HH:MM)")
    temperature: float = Field(description="мҳҲмғҒ мҳЁлҸ„ (В°C)")
    precipitation: float = Field(description="к°•мҲҳ нҷ•лҘ  (%)", ge=0, le=100)


class DetailedWeather(BaseModel):
    """мғҒм„ё лӮ м”Ё м •ліҙ"""
    city: str = Field(description="лҸ„мӢңлӘ…")
    location: Coordinates = Field(description="лҸ„мӢң мўҢн‘ң")
    current: WeatherData = Field(description="нҳ„мһ¬ лӮ м”Ё")
    forecast: list[ForecastItem] = Field(description="мӢңк°„лі„ мҳҲліҙ (6мӢңк°„)")


@mcp.tool()
def get_detailed_weather(city: str) -> DetailedWeather:
    """лҸ„мӢңмқҳ мғҒм„ё лӮ м”ЁлҘј мЎ°нҡҢн•ңлӢӨ"""
    return DetailedWeather(
        city=city,
        location=Coordinates(latitude=37.5665, longitude=126.9780),
        current=WeatherData(
            temperature=22.5, humidity=45.0,
            condition="л§‘мқҢ", wind_speed=5.2
        ),
        forecast=[
            ForecastItem(time="15:00", temperature=24.0, precipitation=10.0),
            ForecastItem(time="18:00", temperature=21.0, precipitation=30.0),
            ForecastItem(time="21:00", temperature=18.5, precipitation=60.0),
        ],
    )

Optional н•„л“ңмҷҖ кё°ліёк°’

from typing import Optional


class SearchResult(BaseModel):
    """кІҖмғү кІ°кіј"""
    title: str = Field(description="м ңлӘ©")
    url: str = Field(description="URL")
    snippet: Optional[str] = Field(default=None, description="мҡ”м•Ҫ (м—Ҷмқ„ мҲҳ мһҲмқҢ)")
    score: float = Field(default=0.0, description="кҙҖл ЁлҸ„ м җмҲҳ (0~1)")
    tags: list[str] = Field(default_factory=list, description="нғңк·ё лӘ©лЎқ")
TIP

Pydantic лӘЁлҚёмқҳ docstring("""лӮ м”Ё м •ліҙ""")мқҖ JSON Schemaмқҳ descriptionм—җ нҸ¬н•ЁлҗңлӢӨ. лӘЁлҚё мһҗмІҙм—җ лҢҖн•ң м„ӨлӘ…мқ„ л°ҳл“ңмӢң мһ‘м„ұн•ҳмһҗ. Fieldмқҳ ge, le л“ұ м ңм•Ҫ мЎ°кұҙлҸ„ JSON Schemaм—җ л°ҳмҳҒлҗҳм–ҙ нҒҙлқјмқҙм–ёнҠём—җм„ң мӮ¬м „ кІҖмҰқмқҙ к°ҖлҠҘн•ҙм§„лӢӨ.

TypedDict мӮ¬мҡ©

PydanticліҙлӢӨ к°ҖлІјмҡҙ лҢҖм•ҲмңјлЎң TypedDictлҘј мӮ¬мҡ©н•  мҲҳ мһҲлӢӨ. лі„лҸ„ мқҳмЎҙм„ұ м—Ҷмқҙ Python н‘ңмӨҖ лқјмқҙлёҢлҹ¬лҰ¬л§ҢмңјлЎң кө¬мЎ°лҘј м •мқҳн•ңлӢӨ. 

from typing import TypedDict, NotRequired


class LocationInfo(TypedDict):
    latitude: float
    longitude: float
    name: str
    altitude: NotRequired[float]  # м„ нғқм Ғ н•„л“ң


@mcp.tool()
def get_location(address: str) -> LocationInfo:
    """мЈјмҶҢмқҳ мўҢн‘ңлҘј мЎ°нҡҢн•ңлӢӨ"""
    return LocationInfo(
        latitude=37.5665,
        longitude=126.9780,
        name="м„ңмҡё",
    )

  • лҹ°нғҖмһ„ лҚ°мқҙн„° кІҖмҰқ м§Җмӣҗ
  • Field(description=...)мңјлЎң мғҒм„ё л©”нғҖлҚ°мқҙн„°

  • JSON Schema мһҗлҸҷ мғқм„ұ (мҷ„м „н•ң м§Җмӣҗ)
  • мқҳмЎҙм„ұ: pydantic нҢЁнӮӨм§Җ н•„мҡ”
TypedDict

  • н•„л“ң м„ӨлӘ… л¶Ҳк°Җ (мҶҚм„ұлӘ…л§Ң л…ём¶ң)
  • мӨ‘мІ© к°ҖлҠҘн•ҳлӮҳ Union м§Җмӣҗ м ңн•ңм Ғ
  • JSON Schema мғқм„ұмқҙ лӢЁмҲңн•Ё

WARNING

TypedDictлҠ” лҹ°нғҖмһ„ мң нҡЁм„ұ кІҖмҰқмқ„ мҲҳн–үн•ҳм§Җ м•ҠлҠ”лӢӨ. мһҳлӘ»лҗң нғҖмһ…мқҳ к°’мқ„ л„Јм–ҙлҸ„ м—җлҹ¬к°Җ л°ңмғқн•ҳм§Җ м•ҠмңјлҜҖлЎң, лҚ°мқҙн„° л¬ҙкІ°м„ұмқҙ мӨ‘мҡ”н•ң лҸ„кө¬м—җлҠ” Pydanticмқ„ мӮ¬мҡ©н•ҳмһҗ.

JSON Schema мһҗлҸҷ мғқм„ұ

FastMCPк°Җ Pydantic лӘЁлҚём—җм„ң outputSchemaлҘј мғқм„ұн•ҳлҠ” мӣҗлҰ¬лҘј мқҙн•ҙн•ҳл©ҙ, мҠӨнӮӨл§ҲлҘј м»ӨмҠӨн„°л§Ҳмқҙм§•н•ҳкұ°лӮҳ л””лІ„к№…н•  л•Ң мң мҡ©н•ҳлӢӨ.

model_json_schema() кІ°кіј л¶„м„қ

from pydantic import BaseModel, Field
import json


class WeatherData(BaseModel):
    """лӮ м”Ё м •ліҙ"""
    temperature: float = Field(description="м„ӯм”Ё мҳЁлҸ„")
    humidity: float = Field(description="мҠөлҸ„ (%)")
    condition: str = Field(description="лӮ м”Ё мғҒнғң")
    wind_speed: float = Field(description="н’ҚмҶҚ (m/s)")


# JSON Schema м§Ғм ‘ нҷ•мқё
schema = WeatherData.model_json_schema()
print(json.dumps(schema, indent=2, ensure_ascii=False))

model_configлҘј мӮ¬мҡ©н•ҳл©ҙ JSON Schemaмқҳ лҸҷмһ‘мқ„ м„ёл°Җн•ҳкІҢ м ңм–ҙн•  мҲҳ мһҲлӢӨ:

from pydantic import ConfigDict


class StrictWeather(BaseModel):
    """м—„кІ©н•ң лӮ м”Ё лӘЁлҚё вҖ” м¶”к°Җ н•„л“ңлҘј кёҲм§Җн•ңлӢӨ"""
    model_config = ConfigDict(
        json_schema_extra={
            "examples": [
                {
                    "temperature": 22.5,
                    "humidity": 45.0,
                    "condition": "л§‘мқҢ",
                    "wind_speed": 5.2,
                }
            ]
        }
    )

    temperature: float = Field(description="м„ӯм”Ё мҳЁлҸ„")
    humidity: float = Field(description="мҠөлҸ„ (%)")
    condition: str = Field(description="лӮ м”Ё мғҒнғң")

@mcp.tool()

async def call_with_structured_output(session: ClientSession):

from typing import Annotated, Literal, Union
from pydantic import BaseModel, Field


class SuccessResult(BaseModel):
    """м„ұкіө мқ‘лӢө"""
    status: Literal["success"] = "success"
    data: dict = Field(description="мқ‘лӢө лҚ°мқҙн„°")
    elapsed_ms: float = Field(description="мІҳлҰ¬ мӢңк°„ (ms)")


class ErrorResult(BaseModel):
    """м—җлҹ¬ мқ‘лӢө"""
    status: Literal["error"] = "error"
    error_code: str = Field(description="м—җлҹ¬ мҪ”л“ң")
    message: str = Field(description="м—җлҹ¬ л©”мӢңм§Җ")
    retry_after: int | None = Field(default=None, description="мһ¬мӢңлҸ„ лҢҖкё° мӢңк°„ (мҙҲ)")


# Discriminated Union: status н•„л“ңлЎң нғҖмһ…мқ„ кө¬л¶„н•ңлӢӨ
ApiResponse = Annotated[
    Union[SuccessResult, ErrorResult],
    Field(discriminator="status"),
]


@mcp.tool()
def fetch_data(endpoint: str) -> ApiResponse:
    """мҷёл¶Җ APIм—җм„ң лҚ°мқҙн„°лҘј к°Җм ёмҳЁлӢӨ"""
    if endpoint == "/health":
        return SuccessResult(
            data={"healthy": True},
            elapsed_ms=12.5,
        )
    else:
        return ErrorResult(
            error_code="NOT_FOUND",
            message=f"м—”л“ңнҸ¬мқёнҠё '{endpoint}'лҘј м°ҫмқ„ мҲҳ м—ҶлӢӨ",
            retry_after=None,
        )

STOCKS = {
    "005930": {"name": "мӮјм„ұм „мһҗ", "price": 72000, "change": 1500, "volume": 15234567},
    "000660": {"name": "SKн•ҳмқҙлӢүмҠӨ", "price": 185000, "change": -2000, "volume": 3456789},
}

@mcp.tool()
def get_stock_price(symbol: str) -> StockPrice:
    """мў…лӘ©мқҳ нҳ„мһ¬ к°ҖкІ©мқ„ мЎ°нҡҢн•ңлӢӨ"""
    data = STOCKS.get(symbol)
    if not data:
        raise ValueError(f"мў…лӘ© мҪ”л“ң '{symbol}'мқ„ м°ҫмқ„ мҲҳ м—ҶлӢӨ")

    return StockPrice(
        symbol=symbol,
        name=data["name"],
        current_price=data["price"],
        change=data["change"],
        change_percent=round(data["change"] / (data["price"] - data["change"]) * 100, 2),
        volume=data["volume"],
        market_status=MarketStatus.OPEN,
        timestamp=datetime.now().isoformat(),
    )


@mcp.tool()
def get_portfolio(symbols: list[str]) -> PortfolioSummary:
    """нҸ¬нҠёнҸҙлҰ¬мҳӨ мҡ”м•Ҫмқ„ мЎ°нҡҢн•ңлӢӨ"""
    holdings = []
    total_value = 0.0
    total_cost = 0.0

    for symbol in symbols:
        price_info = get_stock_price(symbol)
        holdings.append(price_info)
        total_value += price_info.current_price * 100
        total_cost += (price_info.current_price - price_info.change) * 100

    total_profit = total_value - total_cost
    profit_rate = round((total_profit / total_cost) * 100, 2) if total_cost > 0 else 0.0

    return PortfolioSummary(
        total_value=total_value,
        total_profit=total_profit,
        profit_rate=profit_rate,
        holdings=holdings,
    )


@mcp.tool()
def place_order(symbol: str, quantity: int, order_type: str) -> TradeResult:
    """мЈјл¬ёмқ„ мӢӨн–үн•ңлӢӨ (мӢңл®¬л Ҳмқҙм…ҳ)"""
    stock = STOCKS.get(symbol)
    if not stock:
        return TradeFailure(
            reason=f"мў…лӘ© мҪ”л“ң '{symbol}'мқҙ мЎҙмһ¬н•ҳм§Җ м•ҠлҠ”лӢӨ",
            can_retry=False,
        )

    if quantity > 1000:
        return TradeFailure(
            reason="1нҡҢ мөңлҢҖ мЈјл¬ё мҲҳлҹү(1000мЈј)мқ„ мҙҲкіјн–ҲлӢӨ",
            can_retry=True,
        )

    return TradeSuccess(
        order_id=f"ORD-{datetime.now().strftime('%Y%m%d%H%M%S')}",
        executed_price=stock["price"],
        quantity=quantity,
    )
